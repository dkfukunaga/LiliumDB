Pager

fetchPage(PageNum)      — get a page, pinning it in the buffer pool
unpinPage(PageNum)      — release the pin after you're done
newPage()               — allocate a new page, returns pinned page
deletePage(PageNum)     — mark a page for deletion
flushPage(PageNum)      — force a specific dirty page to disk
flushAll()              — flush all dirty pages

CA 70 CA FE 4C 44 42 00 — magic bytes, LDB\0 visible in ASCII sidebar
00 00                   — flags, none set
01                      — version major 1
00                      — version minor 0
02 00 00 00             — num_pages = 2 (little-endian)
01 00 00 00             — catalog_root = page 1
FF FF FF FF             — free_list = INVALID_PAGE
00 00 00 00 31 2D 0B 00 — file_created timestamp
00 00 00 00 31 2D 0B 00 — last_modified timestamp (same, just created)

24 bytes of 00 reserved padding
remaining 4032 bytes zeroed, compressed to *
